\subsection{Intro to LaTex}
LaTex is a language for writing and formating serious stuff in a reproducible way. Unlike Microsoft Word, you will not see the text appear in the document directly when writing it. Instead you have to run the code in order to "build" your output pdf.
\subsubsection{Installing LaTex}
As any other "programming language", you have to install it before you can use it. Depending on your operating system it might already be installed, but you can also get it from \href{https://www.latex-project.org/get/#tex-distributions}{here}.
\subsubsection{LaTex in VSCode}
\Ac{vscode} is a widely used \ac{ide}. It can be extended by installing extensions from community members or organizations. For writing LaTex code you can use the \href{https://open-vsx.org/vscode/item?itemName=James-Yu.latex-workshop}{LaTex Workshop} extension by James-Yu.
